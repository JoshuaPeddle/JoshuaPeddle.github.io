Longest Common Subsequence


    What is the Longest Common Subsequence problem? Wikipedia defines "The Longest common subsequence problem is the problem of finding the longest subsequence common to all subsequence in a set of sequences (often justtwo sequences)." Focusing on the two sequence problem, consider the following example. Given two strings, GATCATC and CTACGTC, the longest common subsequence would be ACTC. This example isn't too hard to solve by inspection but with longer strings solving through inspection is not feasible.
