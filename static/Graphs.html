Overview of Graphs




Traversals


     Visiting each vertex in a graph is the basis of many graph algorithms. Such algorithms are called traversals. BFS (Breadth First Search) and DFS (Depth First Search) are the most popular and usually learned together. Upon initial inspection you'll notice both are similar in implementation and technically accomplish the same thing. (Visit every vertex in a graph) The difference lies in the order that the algorithm considers the vertices.

     BFS selects vertices closer to the source first. This is useful for applications where the target has a higher chance of being closer to the source.



     DFS moves away from the source by selecting descendants before siblings. If the target tends to be far away for the source BFS will be inefficient because it will spend time checking closer vertices first.
