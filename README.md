# JoshuaPeddle.github.io
Personal Website hosted using GitHub pages. Here's a [Link](https://joshuapeddle.github.io/).

This is a static website build using HTML and CSS.

[MathJax](https://www.mathjax.org/) is used to render latex style exquasions in the browser.

[Manim community](https://docs.manim.community/en/stable/reference/manim.mobject.graph.Graph.html) was used to create the images and videos. Links to the source code for animations will be linked at the bottom of the post pages.
